之前与 hush 君交流时，得到了几个命题作业，我内心真正的苦痛便藏于其中。

1. 浅薄之患，对应于与人比较时产生的自卑、羞耻，与完美自己比较时产生的内耗，看看大家都有些什么不一样的解法。
2. 内向的人和外向的人的区别都有撒。
3. 如何看待死亡，或者经历过哪些与死亡有关的故事。
4. 大家都是怎么找到内心真正想做的事情的呢。
5. 最爱吃的一道菜是撒，背后有怎样的回忆。

# 依依

1. 没有看法，不自卑，不羞耻，只有羡慕
2. 内向的人主要靠自己独处或者和最亲密的朋友玩获得能量，外向的人喜欢社交
3. 大家都要死的，或早或晚，坦然接受
4. 喜欢的事就是内心真正想要做的事吗，那当然是让自己快乐的事情
5. 喜欢吃的太多，没有最爱的

# 国政

1. 浅薄之患对我来说好像是阶段性的，目前觉得没有刻意的解法，更像是不得不直接面对，「受着」，然后依靠行动将注意力转移回自身。我觉得浅薄之患本质是对自我认知的偏移，而这种偏移是很正常的。我接触互联网后，觉得有个说法很有道理，即很多行业其实是由 10% 的人在撑着，其余的人要么是在包装，要么是在做着一些螺丝钉的工作，可是我们很容易只看到那 10% 的人。再者，即便真正出类拔萃的是这 10% 的人，也不妨碍我们在能养活自己的情况下也活得很开心——生活还是为了生活本身，找多一些生活的支撑点，比较只是其中一种，这可能需要一些不同的经历、体验来沉淀自身。

2. 内向的人和外向的人在我看来只是表象，行为模式有所不同，对个人发展方向有所影响，但不绝对。内向的人可能被认为更容易独处、敏感、专注于做某件事情，外向的人则可能更容易被认为要在与人交往中获取能量，以及发展。

3. 目前，我认为面对死亡的方式是，在死亡到来以前「练习死亡」——具体来说可以通过阅读、哲思、冥想、讨论等方式进行，但在死亡发生时（主要指身边的亲朋好友），我认为看待死亡的方式就是「受着」，只能去感受经历，这个阶段其实很难静下心去阅读什么，去思考什么，其实就是感受着，视情况决定维持好个人最低的生理限度，十分难过的情况下保证好「饿了好好吃好吃的东西，困了好好大睡一觉，难受了找好朋友纵情地倾诉」。

4. 我觉得我还没找到内心真正想做的事情，但我觉得「在实践中寻找」是一个很好的指导思想，它会衍生出很多可能，即便最终没找到，行动也会消解一些不断反刍「我真正想做的是什么？」带来的迷茫感。

5. 还没想到……

# 诗涛

1. 停止与人比较，不幻想完美自己。别人展现出来都是别人想展现的样子，完美自己也不存在（或者是综合了所有别人家孩子的优点）。如你所说，光明的另一面同样真实，不完美的自己也是自己，黑塞的感觉来了。自信的人必须对自己黑暗的那面同样自信，不然要人格分裂。
2. 内向外向是比较的结果。
3. 向死而生，死亡的存在让当下一切都有意义。如果人不会死，那时间、当下都没有意义。
4. 排除法。
5. 我居然说不出来，我过得太粗糙了，我要精致些，多思考下这些问题。

这个4我想补充一下。排除法是最低门槛的方法，但并不一定最优。因为我一直在思考「成为我自己」这个话题，通过自己的觉察觉知也可以知道自己喜欢的事情，并通过尝试去确认。后续真正做想做的事情是另一个话题，关于勇气等等。

# 小二

1. 对自我的认知，定位，与实际的自己有差异，不接受真实的自己。没有享受过真正无条件的爱与接纳。解决方案：认识信仰知道自己是独一无二的存在。并且在永恒中的审判，善与恶，并不一定与我们人现在的评判标准是一样的。

2. 可能内向的人，防御程度更高一些，只有那么极少几个人可以让他敞开心扉畅所欲言吧。而外向的，愿意聊更多的人？

3. 死亡不是终点，但人死后虽有灵魂，灵魂依然是人，不会变成鬼也不会变成神。人活着所做所思所说的一切，会决定死后的结局，一定会有最终的审判。所谓的亡灵保佑，在基督教里面是否定的。孝敬父母，在世得长寿。今生事，今生毕，不要想着死后多烧些纸再说了，活着不对父母好点，死了想让亡魂来保佑你，凭啥呀。人死后，针对这个人的结局，就已经定了，没有办法了，所以，活着，才有机会和一切可能。好好活着。

4. 想做的事，不一定是正确的事。在基督教里面，我们更鼓励大家，不管在什么环境下，尊重信仰，尊重法律，尊重自然，尊重原则。要去做正确的事虽然很难，但每一步要走踏实，知道未来，上帝必有奖赏。

5. 没有固定的喜欢吃的，但我做的菜能让大家吃满意，吃饱并且光盘，我会很开心。

# X 兄弟

1. 做好自己就行，不羡慕别人，不攀比
2. 与人交往方面，外向的人更喜欢主动输出，与人互动，内向的人则多与自己沟通，较少与外人互动，或互动的效果也不好
3. 信奉一句话：除了生死，其他的都是小事。我的爷爷、奶奶、外公、外婆均已去世，我弟弟和我妈妈也已经去世，个人感触是看着至亲之人离世是最悲痛的，比如我弟弟的离世就是家人最悲痛的时候，没有亲身经历过的话应该是无法感同身受的。
4. 不断尝试，不断询问自己，自己擅长什么，对什么感兴趣，然后大胆的尝试
5. 好吃的菜有很多，第一印象是北京一家餐馆的麻辣香锅，后来也吃过他们家的其他分店的菜，只是再也找不到那么好吃的了。

# 阿木狗主

1. 最近主要是不断反刍阿木还在的时候我少做了些什么，自责、悔恨、愧疚虽然会消耗心力，但并不需要解决什么，只是我思念阿木的一种方式。那个“完美自己”，也只是我想象的另一个平行时空，暂时没有办法停止想象。不过也许会随着时间推移，慢慢少想一点吧。
2. 内向、外向我觉得区别不大，主要表现在向内求、向外求的比例不同。
3. 让我接受自己的死亡，是很轻松的，可是接受阿木的死亡，是煎熬的。
4. 我也不知道，长远来看我没有什么规划，所以主要在日常生活中去做想做的一些小事。
5. 喜欢吃的都是老家我奶奶我妈妈做的每年吃的菜。

再补充一部分，如下。

比较是认识差异的工具。人们除了通过比较会内耗以外，有时候也会获得优越感。比较本身只是工具，更应该重视的是情绪背后的需求。举个例子，人的身高有高矮之分，比较大量样本后，发现自己低于平均值，但依然可以选择接受自己的特点，不去贬低自己。又比如，比较样貌后，也可以得到很多结论，但审美标准并不需要所有人一致。又比如，技术方面，好多大神很厉害，比较之后觉得自己真菜，依然可以选择不责怪自己，而是一步一步走自己的路。总地来说，其实真的不用对任何事都保持自信，那样太紧绷了一些。

小福：这连着五个问题好长，一个一个看看。关于浅薄之患上，结合生活中的小例子，每个人都不完美，比如自己，有时候看到别人长的漂亮，自己却没有，自己就会想到，有些是可以改变的，比如人的状态。有些也是不能改变的，比如人的五官布局等（用现代医美可以叫整容，但是那个先不列入考虑，基本还不到那种地步）然后就可以想，往着更好的人或者更完美的自己迈进，可以怎样去做。然后去找自己能改变的部分去入手。同时也告诉自己，每改变一点自己都在变得更好。通过与过去的自己进行对照，感受上就会好很多。而不是一直盯着终点看，去想我为什么还没到终点。在与其他人交流的过程中，也能看到很多人天赋异禀，或精力充沛，或神智过人，交流之中宛如机器人计算一样。会有羡慕和赞叹的心理，甚至是惊掉下巴，这也特别正常。如果能有和这种人交流的机会，可以大胆发问，或者在他的言行举止中，自己去做一些思考和启发。看那些地方能为我所用，促进自己变好。还有的也是，多见，慢慢见的越来越多，越感知到人外有人天外有天，慢慢也可能会出现祛魅或者见到更加真实的这种人。加上也能发现有这种厉害的人，基本上也是在某一方面，也没有有像上帝一样无所不能的人，有的话，那就叫神了。
