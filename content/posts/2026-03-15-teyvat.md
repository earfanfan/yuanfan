我最近总感觉穿越到了一个平行世界里，对时间的感觉不再是线性的，因为每天都会无数次回到那一天。

<!--more-->

我最近也有在积极地生活，给每个周末都找点事做，给生活添点盼头。上周我去深圳看了蔡依林的演唱会，是人生中第一次看演唱会呀。昨天我去菜市场买了饺子皮，回来包了50个饺子，一半猪肉芹菜馅，一半虾仁韭菜馅的，也是人生中第一次包饺子呀，因为捏合饺子很认真，快两小时才包完。下个周末去荆门，又会和依依一起穿汉服踏青、赏花、拍照，希望樱花谷的樱花们能好好扛过下一周的雨。除了这些，还有一个长期的计划，执行一个为期三个月的减重计划，从64公斤减到58公斤，无论成功或失败，这个尝试的过程都会写一篇博客好好记录下来。啊，我真得有在努力生活呀，虽然现在对这个世界还提不起撒兴趣，将来应该会恢复的吧。

最近脑子很空，好像观察和思考的能力彻底停滞，可能有部分神经断了吧。每晚睡前会翻翻手机相册、刷刷 B 站视频，上周和本周都有刷到一些视频让我感到很治愈，也许低谷时期让自己的脑子成为别人的跑马场也挺好的吧。

# 一

以下整理的台词源自 B 站 up 主孙悟穷剪辑的[圣僧一开口，就是满分作文!治愈解压，听完烦恼全消失](https://www.bilibili.com/video/BV1wfHYzBEva/?spm_id_from=333.1387.homepage.video_card.click&vd_source=7d5f3b7e130609b6f2de8f9e7eb95118)。最开始是刷到同一个 up 主剪辑的[凡僧与圣僧的差距](https://www.bilibili.com/video/BV1ravAzDE82/?spm_id_from=333.1387.homepage.video_card.click&vd_source=7d5f3b7e130609b6f2de8f9e7eb95118)，然后又看了 B 站关联推荐的[凡僧与圣僧的差距(二)](https://www.bilibili.com/video/BV1hQndzmEBb/?spm_id_from=333.1387.homepage.video_card.click&vd_source=7d5f3b7e130609b6f2de8f9e7eb95118)和段首提到的视频。其内容主要取自 TVB 版的《西游记》，弹幕吐槽这版唐僧自己都可以写一部经书出来。

下面最治愈我的是“因果循环，不可逆转”，原来至亲至爱的老病死这种事，真得除了接受以外别无办法。

1. 

  + 路人问：你们是从哪里来？要到哪里去？
  + 唐僧答：从烦恼的地方来，要去没有烦恼的地方。

2. 

  + 唐王问：此去西天几时才能回来？
  + 唐僧答：松树枝头向东之日，亦是三藏归来之时。

3. 

  + 悟空问：我们取到真经之后，是不是很自然的就成佛了？
  + 唐僧答：成佛非自然，自然便成佛。
  + 另一个悟空说：只要一片赤诚，灵山就在你脚下。

4. 

  + 唐僧问：你们知不知道我们要取的是什么经？
  + 沙和尚答：我知道，师父，是涅槃经，大般若经以及华严经。
  + 唐僧说：佛祖要我们取经，所谓经者乃是经历人生的意思

5. 

  + 唐僧说：爱别离、怨憎会、求不得，乃是人生之苦。荣华富贵只不过是镜花水月，功名利禄犹如海市蜃楼，贤妻美妾好比骷髅白骨。一切如梦幻泡影，如流星飞石，得失生死，只不过通通是黄粱一梦。

6. 
  + 妖精说：你看我们每个都长得这么美，难道你一点也不心动吗？
  + 唐僧答：所谓色相，皆属虚幻。
  + 妖精问：虚幻？你看不知道有多真实呢。你看我手滑不滑 ，我们个个都像是明珠闪闪。我就不信你一点也不心动吗？
  + 唐僧答：色相好比纯净定珠本来无色，红光来照则遍珠皆红，绿光来照则遍珠皆绿，红绿齐照则遍珠红绿。因宝珠体性本空，虽百千万亿色相相加，包容如故，然色即是空。

7. 

  + 百姓甲：大圣爷在上，希望你保佑小民大发横财，逢赌必赢。
  + 百姓乙：小民是卖棺材的，希望大圣爷你保佑小民的生意兴隆。
  + 百姓丙：小民是行医的，希望大圣爷显灵显性，让本村来场大瘟疫。
  + 唐僧说：世人愚昧无知，并非你三言两语、一朝一夕可以使他们大彻大悟的。
  
8. 

  + 唐僧说：佛家导人向善，有所谓的顿悟和渐悟。佛祖要你们经历三三之难、九九之劫，就是要你们慢慢领悟人生、远离苦海、脱离红尘的意思。

9. 
  + 通臂猿猴说：我现在爱怎么样就怎么样，我可以领万妖取西经，三界五行无仙无佛可以阻拦我！
  + 唐僧说：人生之苦来自贪嗔痴恨，你扪心自问，自你脱离仙籍、加入万妖国，你可曾有一天开心过？
  
10. 
  + 万妖女王说：夫君大仇未报，我至死不休。
  + 唐僧说：你满手血腥、行凶作恶不外是为了替通背猿猴报仇。因为你执着太深，自然魔障重重，你一直所见到的通背猿猴，只不过是你自己的心魔。

11. 
  + 悟空吐槽：这些妖怪把舌头伸出来捡东西吃，成何体统！
  + 唐僧说：每个人都有他的饮食习惯，蜘蛛吃飞虫就等于牛吃草一样，你又怎么可以那么蛮横无理呢。

12. 

  + 另一个八戒说：唉哟，师父，你这是怎么了，你这不是把别人的棺材抬到自己家里哭嘛。他伤他的子民，关你什么事！

13. 
  + 唐僧说：眼前的人命你都不去救，就算让你取得大乘佛法又有何意义。

14. 

  + 老和尚说：老衲这座观音禅院，堪称是方圆几千里之内最豪华最宏伟的寺庙啊。
  + 唐僧说：观音得道，早已超脱生死轮回。我等学佛应该无欲无求，又何必追求那些腐朽之物呢。只要心存敬畏，加上一颗向佛之心就够了。

  + 沙和尚说：袈裟明明是你的，你不要因为徒儿顽劣就把袈裟双手送给别人呐。
  + 唐僧说：如果为师把袈裟交给他们，说不定他们向佛之心会更加坚定，那到时候我们造的福德岂不是更大。

15. 

  + 妖精说：你身为圣僧，竟然为了我这妖精说谎话。
  + 唐僧说：你本性不坏，只不过是一念之差，才会误入妖道。如果贫僧说了这个谎话，可以使你回头是岸的话，那又有何不可呢。

16. 
  + 唐僧说：悟空，你一日拜我为师，终生都是我的徒弟，为师可没这么容易放弃。你要成亲，为师就做你的证婚人。你要生孩子，为师就帮你带孩子。假如你要吃人，就先吃了为师。
  + 悟空说：你的肉怎么好吃呢，你吃素的，一定是淡而无味。
  + 唐僧说，佛祖当年割肉喂鹰。观音菩萨也曾为救一个妇人而舍弃成佛的机会。地藏王也曾许下宏愿，地狱不空、誓不成佛。我唐三藏，今日又何忍见悟空再走错路，而受到万劫不复之苦。

17. 

  + 沙和尚对唐僧说：你的亲爹才刚刚死了啊，你还想着吃？！
  + 唐僧对他爹说，爹，为人子者，刚刚和父母重聚，又要面临生离死别，我怎么会不伤心。生死循环，有因有果，不能强求，也不能逆转。只有坦然面对，才可以消除恐惧，脱离苦海。
  + 悟空说：只要没恐惧就没痛苦了，师父。
  + 唐僧说：心无挂碍，无挂碍故，无有恐怖，能除一切苦。既然心里面没有恐惧，又怎么会有痛苦。

18. 
  + 旁白：话说唐三藏他知道这并非猪八戒的骨灰之后，他仍然决心要拯救这个徒弟。
  + 沙和尚说：师父，这个根本不是二师兄，你还念什么经啊。
  + 唐僧说：众生皆可成佛，人畜皆为众生，众生既然平等，有何区分之别呢。

19. 

  + 唐僧梦里象鼻子精说：我自从听了圣僧的话之后，我的心，哇，好平静啊。谢谢圣僧，谢谢圣僧。
  + 唐僧乐得大笑不止：世人尽渡，渡尽世人。

# 二

源自 B 站 up 主岭南城北剪辑的[原神中那些治愈的台词](https://www.bilibili.com/video/BV125411Q75x?spm_id_from=333.788.videopod.sections&vd_source=7d5f3b7e130609b6f2de8f9e7eb95118)。

下面这些台词仅看文字未免觉得单调，听到声音就能更好地接收到每个角色传递的能量，这些似乎重新唤醒了一分我的中二魂，我更少问自己阿木去了哪里，而是开始相信阿木依然存在。

1. 不知道如何向前的话，总之先迈出第一步，后面的道路就会自然而然地展开了。

2. 只要坚信自己的道路，就无所谓天气是晴是雨。

3. 觉得自己是正确的，就要大声的说出来，坚决地去行动。心有所向。日复一日，必有精进。

4. 遇见有趣的人、有趣的事，心里清楚该往何处迈出下一步，这样就已经获得快乐了。

5. 如果有一天你遇到无法解决的难题，也别认为那就是人生的终点，说不定会有人替你解决困境呢。

6. 最初的鸟儿是不会飞翔的，飞翔是他们勇敢跃入峡谷的奖励。重要的不是强风，而是勇气，是它让你们成为世上最初的飞鸟。

7. 既然神都没有放弃我，我又怎么能放弃我自己呢。

8. 在这个真实的太过分的世界里，每个人都需要仰望星空。

9. 搞得定的事不必烦恼，搞不定的事烦恼也没有意义。去吧！一帆风顺！

10. 不论前路是星辰还是深渊，能向前迈出下一步，都是值得庆贺的事。

11. 当你重新踏上旅途之后，一定要记得，旅途本身的意义。终点并不意味着一切，在抵达终点之前，用你的眼睛多多观察这个世界吧。

12. 有什么烦恼千万不要埋在心里，来和我说说吧。虽然我不是什么事情都在行，但我一定会尽力为你出谋划策的，就算，就算，答案不够可靠，至少我也是你忠实的听众。

13. 如果有一天，连你也陷入了黑暗，便呼我名。三眼五显仙人，魈，听召，前来守护。

14. 有问题就大胆问出来，没有答案就会一直追寻下去，别不好意思，这才是正确的态度，毕竟谁又能保证神就不会犯错呢。

15. 没有任何人有资格剥夺他人的愿望，即便是神，也不行。

16. 总会有地上的生灵，敢于直面雷霆的威光。

17. 我很喜欢枫叶，可惜，枫叶红时，总多离别。

18. 我必须前进，不论有谁阻挡在我面前，在黎明到来之前，必须有人稍微照亮黑暗。

19. 我很信奉摇滚的反抗精神，但我觉得，这不代表我就一定要去张扬个性。我也可以有很正常的生活，可以做缝纫，可以养花养鱼，可以做造型可爱的菜肴，可以喜欢任何事，还有任何人。

20. 做某一件事不代表爱它，不爱一件事，也不意味着必须远离。

21. 无论现在是多精湛的猎手，总还能更进一步，不要因为自满而止步，不要停止学习。我会尽全力与你一起前进，直到终点。

22. 我不会踌躇，也不会辜负。

23. 不要小瞧斗虫啊，在一次次拼搏中，用汗和泪浇灌出更强的自己，这就是人生啊。

24. 我以此剑起誓，必将胜利献给你。

25. 须知一切力量皆有其代价，一千种权力伴随着一千种责任。

26. 本事很重要，不然空有热血只会坏事。

27. 大多数人会因为大风迷眼而止步不前，只有少部分人能够借势而为。

28. 万事皆有代价，既然寻求常人无法企及之事，那就必然付出世人无法想象的代价。

29. 为何人们总是拼命证明，自己的存在比其他人更有价值呢，存在本身就是足够美好的事情了吧。

30. 正是因为无法更改，不可违逆，只能接受，命运才会被称之为命运。

31. 海鸥别叫了，从今天起，我也是漂泊者呀。

32. 我偶尔能感觉到，过去每一年的自己，都存在着，围绕着我，对我有所寄托，你一定也有相同的感觉。所以，哪怕如今你已走上截然不同的道路，也不要否认从前的自己。从前的每一个你都是你脚下的基石，不要害怕过去，不要畏惧与他抗衡。

33. 若知是梦何须醒，不比真如意相会。

34. 现在的我应该有能力保护其他人了，这下子大家都会对我慢慢改观吧。那个倒霉的要命、总是遍体鳞伤的班尼特，终于变成一个可靠的冒险家了。

35. 只要你需要我，我随时都会出现在你的身边。

36. 如果心中没有善恶的标尺，努力就会错了方向，舍本逐末，变成追逐暴利的俗人。

37. 有时明月无人夜，独向昭潭制恶龙。

38. 伙伴！不要忘记，厮杀的时候，绝不能用什么燃烧起来了之类的蠢话来驱策自己。毕竟燃尽之后，除了灰烬什么也不会剩下。无论如何都要活下去，绝对不要倒下，伙伴，我绝不允许你倒下。

39. 阴阳有序，命运无常。死亡难以预测，却也有它的规矩。记住了，无论何时何地，无论有何原因，人都不该轻易挑衅死之一事。唯有认识它、尊重它，才能明白活着的价值。力量若达到极限，接着考验的便是人心。生老病死，天地万象，尽在吾辈。

40. 任何旅途中的旅伴同行，都会有分别的一日。假使有一日不得不同你相别，你在我的记忆中也会如黄金般闪耀。

41. 所谓的永恒，就是不再失去已拥有之物。但从别的角度说，也放弃了得到更多的可能。不过世事总是如此，没人能够得到全部。

42. 浮世景色，百千年依旧。人之在世，却如白露与泡影，虚无。

43. 

  - 就让我把这几百年间的事情讲给你听吧。
  - 几百年间的事，要讲多久？
  - 狐仙的记忆力可是很好的，几百年间的每一个细节我都不会忘记。所以，大概要讲个几百年吧。

44. 变革的时机已经到来，维持了千年的秩序即将被改写。

45. 如果真的有一天，我抵达了理想中的终点，我的身边，还会有你陪伴吗？

46. 风雨犹祝，山海同欢，是承天地之佑。

47. 星移斗转，沧海桑田，烟火人间依旧。功名在我，百岁千秋，毋忘秉烛夜游。今古诸事，激荡中流，宏图待看新秀。

# 三、等烟花

除夕的那个夜晚，依依约我去远处田埂上看烟花，这么多年来第一次离远了看，发现这样看烟花更美。那时候本来想写一篇博客，后来就再也没有办法去写了。我每天都在后悔一些事情……后悔那天应该留在家里陪阿木，或者把阿木裹严实了带去一起看，就是不应该跟阿木分开。

阴差阳错之间，我在录烟花视频的时候用了延时摄影，导致最后视频里看烟花毫无美感，太赶了。于是又找依依给我传了正常速度的版本。

唉，失去阿木对我来说，并不是一种需要完成的课题，也不是要教会我明白什么道理，有些东西得到了就永远不会失去。刚刚是20点30分，天猫精灵又在提醒我要溜阿木了，正好我也坐久了，不知外面雨停了没有，得出去走走呼吸下新鲜空气吧。
